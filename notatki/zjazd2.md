## Wyrażenia listowe (*List comprehensions*)
* https://docs.python.org/3/tutorial/datastructures.html#list-comprehensions

```
a = [ 3, 50, 10 ]

# zwiększenie każdej wartości o 1000
b = [ 1000 + x for x in a ]

# zostawienie tylko wartości mniejszych niż 20
b = [ x for x in a < 20 ]

```
